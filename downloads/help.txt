📝 Tutorial: How to Safeguard Your Code for Reinstallation

Step 1: Initial Purchase
Purchase the code and ensure to save it securely.

Step 2: Code Preservation
Keep your code confidential and avoid sharing it with others.

Step 3: App Reinstallation
If you need to reinstall the app:
Ensure to reinstall it using the correct bundle ID via platforms like Gbox, Boarsign, Esign, etc.
Your code remains valid even after uninstalling and reinstalling the app, as long as it's installed with the correct bundle ID.

Step 4: Transferring to a New Device
If you want to switch the code to a new device:
The price for transferring is 1/4 of the original code price.
You must provide proof of purchase to initiate the transfer.

Step 5: Troubleshooting
If your app stops working due to a software update:
Consider downgrading your device or refrain from updating.

🔒 Important Security Measures:

Save your code in a secure location for future reference.

Step 4: Certificate Installation

Once imported, the Dragon Company certificate will be available for use within the signer application.
